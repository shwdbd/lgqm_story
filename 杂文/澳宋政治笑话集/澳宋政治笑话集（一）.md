# 澳宋政治笑话集（一）
原文地址：https://zhuanlan.zhihu.com/p/28234868

1.澳宋元老、归化民、伪明士绅、伪明百姓商定要见一面。归化民过了约会的时间才到。
——“对不起，我去排队买香肠来着。”
——“什么是排队？”士绅问。
——“什么是香肠？”百姓问。
——“什么是买？”元老问。

2.有个归化民向马千嘱汇报说：“现在大剧院正上演一个剧，里面有您出现，每当您一出场，下面就热烈鼓掌。”马千嘱听了以后非常得意。有一天他买了一张普通票，去看这个剧，他陷入了沉思，忘记了鼓掌，这时旁边有人推了他一把，紧张地说：“哎！你为什么不鼓掌？不要命啦？！政保局在盯着呢！！”

3.某日澳宋举行国庆游行，沿着大街开来了战列步兵、海兵队、掷弹兵、炮兵、黑尔火箭、，破坏力一个比一个大；队列末尾却是两个带公文包的矮个归化民。
在看台上文德嗣惊讶地说：“这两个人破坏力比黑尔火箭还大！他们是什么人？”
政保局的赵曼熊说：“不是我的人。”
席亚洲说：“没见过他们。”
马国务卿说：“他们是企划院的....”

4.一个临高归化民的鹩哥丢了。这是只会骂人的鹩哥，要是落到政保局的手里可糟了。这人便在临高时报上发表了一篇声明：“本人遗失鹩哥一只，另外，本人不同意它的政治观点。”

5.文德嗣和崇祯皇帝在南京开会，休息时间两个人很无聊，就开始比谁的侍卫更忠诚。崇祯先来，他把自己的御前侍卫叫进来，推开窗（外面是5层楼）说： “赵龙，从这里跳下去！”赵龙哭着说：“皇上，请饶了臣下吧，我还有老婆孩子呐。”崇祯被感动了，流着泪说是自己不对，叫赵龙走了，然后抡到文德嗣，他也大声叫自己的保镖李建国。“建国，从这里跳下去！建国二话不说就要往下跳，崇祯一把抱住他说：“你疯了？跳下去会死的！”建国一边挣扎着要跳下去一边说：“放开我，混蛋，我还有老婆孩子呐。”

6.“归化民的儿子能成为元老吗？”
“不可能。元老都有儿子”

7.“假设你在酒吧里，而一个陌生人坐到你的身边并开始唉声叹气，你该怎么做”
“立即去阻止这种反元老院宣传”

8.劳动节游行中，一个非常老的临高士绅举着一个标语：“为了我快乐的童年时代，谢谢你，马千嘱同志”
政保局找到他：“那是什么 ？你在嘲弄我元老院吗？谁都可以看出，当你还是孩子的时候，元老们还没有来到临高”
“确切的说，那就是我感谢他的原因”

9.马千瞩骑着他的小电驴风驰电掣奔入巷子，迎面碰上了文德嗣。文德嗣急呼:“你左我右!”
那天，他们在地上躺了很久。

10.欧洲发生了严重的天花疫情，元老院十分热心的派出了大规模援助队伍，并援助建造了一座带三个大型淋浴室与五座焚尸炉的传染病医院。
